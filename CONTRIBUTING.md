# Contribuer aux Forges

Merci de vouloir contribuer à la souveraineté numérique ! Voici comment fonctionne notre atelier.

## Workflow (Le flux de travail)

1.  **Forkez** le dépôt sur lequel vous voulez travailler.
2.  **Créez une branche** pour votre fonctionnalité (`git checkout -b feature/ma-super-feature`).
3.  **Codez** (en respectant les conventions Rust/Clippy si applicable).
4.  **Commitez** en utilisant l'anglais pour les messages de commit (ex: `feat: add new log parser`).
    * *Note : Nous recommandons de signer vos commits (GPG).*
5.  **Poussez** vers votre fork.
6.  Ouvrez une **Pull Request (PR)** vers notre branche `main`.

## Conventions

* **Langue du Code :** Anglais (variables, commentaires).
* **Langue des Discussions (Issues/PR) :** Français ou Anglais (Bilingue).
* **Formatage :** Utilisez `rustfmt` pour le Rust, `prettier` pour le Web.

## Signaler un Bug

Utilisez l'onglet "Issues" du dépôt concerné. Essayez de fournir :
* Votre OS / Environnement.
* Les étapes pour reproduire le bug.
* Les logs d'erreur.
